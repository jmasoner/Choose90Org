Installation Steps:

Download the Extension

Get started in seconds! Download the Choose90 browser extension and start making your posts more positive right away.

📦 Download Choose90 Extension

Extract the ZIP File

Once downloaded, extract the ZIP file to any folder on your computer. Right-click the file and select "Extract All" - it's that easy!

Remember where you saved it - you'll need this folder in the next step.

Open Extensions Page

For Chrome: Go to chrome://extensions/
For Edge: Go to edge://extensions/

Or: Click the menu (⋮) → Extensions → Manage Extensions

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page.

Load the Extension

Click "Load unpacked" and select the folder where you extracted the extension files.

Pin the Extension (Optional)

Click the extensions icon (puzzle piece) in your browser toolbar, find "Choose90", and click the pin icon to keep it visible.

💡 What is a PWA?

PWA stands for "Progressive Web App" - it's a website that works like a regular app! You can install it on your phone's home screen (just like apps from the App Store or Google Play), but you don't need to download it from a store. It's free, works on both iPhone and Android, and gives you all the features of a native app without the hassle of app store downloads.

❓ Frequently Asked Questions

Which should I use - Extension or PWA?

Desktop users: Use the Browser Extension for seamless integration with Twitter/X, Facebook, and LinkedIn in your browser.

Mobile users: Use the PWA (Progressive Web App) - it works with both mobile browsers and native apps (X app, Facebook app).

Does the extension work with native mobile apps?

No, browser extensions only work in web browsers. For native mobile apps (X app, Facebook app), use the PWA. Compose your post in the PWA, copy it, and paste it into your native app.

Is my data stored or shared?

No! Your posts are processed in real-time and immediately discarded. We don't store, log, or share anything. Your privacy is important to us.

Does it work offline?

The PWA works offline for viewing daily inspiration quotes. For AI rewriting, you'll need an internet connection - but it's worth it for the positive impact!

Is it free?

Yes! Choose90 is completely free. We believe in making positive influence accessible to everyone. Join the movement and help make the internet a better place!

I'm having trouble extracting the ZIP file

If you encounter any issues extracting the file, try extracting it to a simple location like your Desktop or Documents folder. If problems persist, feel free to reach out - we're here to help!
