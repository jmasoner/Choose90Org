Installation Steps:

Download the Extension

Click the button below to download the Choose90 browser extension ZIP file.

📦 Download Extension (ZIP)

Note: Currently available as a developer build. Coming soon to Chrome Web Store!

Extract the ZIP File

Extract the downloaded ZIP file to a folder on your computer. Remember where you saved it!

Open Extensions Page

For Chrome: Go to chrome://extensions/
For Edge: Go to edge://extensions/

Or: Click the menu (⋮) → Extensions → Manage Extensions

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page.

Load the Extension

Click "Load unpacked" and select the folder where you extracted the extension files.

Pin the Extension (Optional)

Click the extensions icon (puzzle piece) in your browser toolbar, find "Choose90", and click the pin icon to keep it visible.

💡 How to Use:

1. Go to Twitter/X, Facebook, or LinkedIn in your browser
2. Start typing a post (at least 50 characters)
3. Look for the "✨ Choose90 can help make this more positive" indicator
4. Click it to see your post rewritten
5. Choose to use the rewritten version or keep your original

The browser extension only works when using social media sites in a web browser. It does not work with native mobile apps (X app, Facebook app). For mobile apps, use the PWA below.

❓ Frequently Asked Questions

Which should I use - Extension or PWA?

Desktop users: Use the Browser Extension for seamless integration with Twitter/X, Facebook, and LinkedIn in your browser.

Mobile users: Use the PWA (Progressive Web App) - it works with both mobile browsers and native apps (X app, Facebook app).

Does the extension work with native mobile apps?

No, browser extensions only work in web browsers. For native mobile apps (X app, Facebook app), use the PWA. Compose your post in the PWA, copy it, and paste it into your native app.

Is my data stored or shared?

No! Your posts are sent to our AI API for rewriting, but they are not stored, logged, or shared. Everything is processed in real-time and immediately discarded.

Does it work offline?

The PWA has offline support for viewing daily inspiration, but AI rewriting requires an internet connection. The browser extension requires an internet connection.

Is it free?

Yes! Choose90 tools are completely free. We believe in making positive influence accessible to everyone.
